# React Quote Generator
This is a simple React app that generates random quotes from an API.

## Used API
[Advice Slip JSON API](https://api.adviceslip.com/)

## How to run locally
1. Clone this repo
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:3000` to view it in the browser
5. Enjoy!
